# 📊 Northwind Sales Analysis Dashboard

## 🚀 Overview
This project analyzes the Northwind dataset to uncover business insights across customers, orders, employees, products, and suppliers.

The analysis is performed using **Excel (EDA + SQL)** and **Power BI (interactive dashboards)** to enable data-driven decision-making.

---

## 🎯 Objectives
- Analyze sales performance across multiple dimensions  
- Identify key revenue drivers  
- Understand customer distribution and behavior  
- Evaluate employee contribution to revenue  
- Analyze product and supplier performance  
- Track trends over time  

---

## 📂 Dataset
The dataset consists of **8 tables**:

- Customers  
- Orders  
- Order Details  
- Products  
- Categories  
- Suppliers  
- Employees  
- Shippers  

### 🔗 Data Model
- Fact Table: `Orders`, `Order Details`  
- Dimension Tables: Customers, Products, Suppliers, Employees, Categories, Shippers  
- Date Table used for time-based analysis  

---

## 🛠 Tools Used
- Excel → Data Cleaning, EDA, Pivot Analysis  
- SQL → Query-based EDA, aggregations, joins, and business question solving  
- Power BI → Data Modelling, DAX Measures, Interactive Dashboards  

---

## 🔍 Data Preparation
- Handled missing values and inconsistencies  
- Created calculated metrics (Revenue, Avg Order Value, etc.)  
- Built relationships between tables  
- Created Date Table for time analysis  
- Defined measures (KPIs) for dashboards

## 💡 Key Insights
- Revenue is concentrated among a small number of products  
- Beverage and Dairy categories contribute the highest revenue  
- Sales show a strong upward trend until a sudden drop (likely incomplete data period)  
- Certain employees generate significantly higher revenue than others  
- Supplier pricing varies widely, impacting profitability  
- Customer distribution is heavily concentrated in specific countries  

---

## 📈 Business Recommendations
- Focus on top-performing products and categories  
- Optimize pricing strategies across suppliers  
- Investigate sudden drops in sales trends  
- Improve performance of low-contributing employees  
- Expand in high-performing geographic regions
